# Changelog

## 0.6.0

- Rename the extension to Pet Viewer for Codex.
- Use the compact `PET` panel title and reduce space around the pet.
- Add six selectable coarse pixel-art backgrounds rendered and animated with Canvas.
- Add Outer Space, Engineering Office, Sunset Overlook, Night City, Server Room, and Secret Treehouse backgrounds.
- Replace all background image assets with responsive procedural Canvas scenes.
- Enrich every procedural scene with layered foregrounds, reflections, props, lighting, and finer pixel detail.
- Add Autumn Forest, Japanese Festival, Tropical Beach, Arcade, Japanese Room, Rainy Café, and Terminal scenes.
- Add Snowy Cabin, animate Terminal commands as live typing, and alphabetize the background picker.
- Redraw Tropical Beach palms with curved segmented trunks, layered green fronds, coconuts, and wind movement.
- Polish every background with richer silhouettes and grounding; redraw broadleaf trees, pines, indoor plants, and branching coral as reusable layered pixel-art elements.

## 0.5.1

- Add a dedicated PET Panel view that can be docked beside the integrated Terminal.
- Add responsive Pet sizing and configurable render scale.
- Add Codex Hooks integration for IDE sessions.
- Aggregate state across multiple workspaces and Codex sessions.
- Preserve existing Hooks configuration during installation and removal.

## 0.4.0

- Add managed Codex App Server integration.
- Reflect running, approval, review, failure, and idle states.

## 0.3.0

- Add display settings, Pet preview, and image format detection.

## 0.2.0
